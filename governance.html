Stollayto STOLL Token Governance

Stollayto implements a decentralized model using community governance through the STOLL token. Use the STOLL token to collect of portion of the platforms revenue and be a part of future Stollayto decisions.

  • 1 CRV for 1 STOLL
  • 10% STOLL tokens reserved for Stollayto team
  • 5% of yield rewards distrubuted to STOLL holders
  • 1 STOLL token is worth 1 vote
  • 3 day timelock to vote on a proposal
  • STOLL token owner and delegate voting rights to another address

Governance Proposal Process

Our proposal model is based upon Uniswap.

1.Idea Check

Detemines if there is sufficent will to make changes. Majority vote will move to the next phase.

2.Consensus Check

Feedback from the Idea Check phase is used towards a formal discussion. Majority vote will being the development of code and move the proposal to the next final phase.

3.Governance Proposal

Developed code is on-chain. Upon majority vote the code will be automatically depoyed and utilized.
