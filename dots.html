Dots and Boxes

Interactive web application to play three player dot and boxes. Players either create a new game and generate a join code or enter a join code to join an existing game. Players take turns drawing lines, if they complete a box with the line drawn they win that box and receive an additional turn. The goal of the game is to win the greatest number of boxes.

Technologies: HTML, CSS, JavaScript, Socket IO
