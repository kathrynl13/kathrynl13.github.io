Audio Visual Phone Unlock

Audio Visual Phone Unlock is a unique take on one way to unlock a phone. User hears an audio prompt describing a hand gesture expected to unlock the phone. User must allow camera access and perform the hand gesture. Harnessing the power of machine learning from the Teachable Machine (https://teachablemachine.withgoogle.com/) we can detect if the user performed the gesture correctly, if the gesture was performed correctly access is granted.

Technologies: HTML, Bootstrap, CSS, JavaScript
